🚀 COMO USAR
═══════════════════════════════════════════════════════════

1. INSTALAÇÃO
   npm install

2. CONFIGURAÇÃO
   - Criar projeto Supabase
   - Executar schema.sql
   - Criar buckets storage
   - Configurar Google OAuth
   - Preencher .env.local

3. DESENVOLVIMENTO
   npm run dev

4. BUILD
   npm run build

5. DEPLOY
   Veja DEPLOYMENT.md
